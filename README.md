# Portfolio Risk Analytics Dashboard

An end-to-end financial analytics platform built using Python, Excel, SQL, and data visualization tools to evaluate portfolio performance, risk exposure, and investment decision-making.

## Tech Stack
- Python
- Excel
- SQL
- Power BI
- yfinance
- pandas
- plotly

## Features
- Portfolio performance tracking
- Risk analytics
- Benchmark comparison
- Sector allocation analysis
- Portfolio optimization
- Automated reporting
